# PM2

## 常用命令

:::tip

详情文档：<https://pm2.keymetrics.io/docs/usage/pm2-doc-single-page/>
:::
```bash
1. pm2 start <file.js> 启动进程

2. pm2 stop [<name> | <pid>] 停止进程

3. pm2 restart & reload [<name> | <pid>] 重启进程

4. pm2 delete [<name> | <pid>] 删除进程

4. pm2 ls 列出所有进程

┌────┬────────────────────┬──────────┬──────┬───────────┬──────────┬──────────┐
│ id │ name               │ mode     │ ↺    │ status    │ cpu      │ memory   │
```
